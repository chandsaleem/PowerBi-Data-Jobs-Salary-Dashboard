An **interactive Power BI dashboard** analyzing the global **Data Jobs market**, focusing on salary insights, job demand, hiring platforms, and remote work availability.

This project explores real-world job posting data to uncover trends in the **data analytics job market** and provide insights useful for job seekers and analysts.

---

# 📌 Project Overview

The goal of this project is to analyze **global job postings in the data industry** and identify patterns in:

- Salary distribution  
- Job demand trends  
- Hiring platforms  
- Work-from-home opportunities  
- Job types (Full-time, Contract, etc.)  
- Geographic distribution of jobs  

The dashboard allows users to **interactively explore job market insights** and drill down into specific job titles such as **Data Analyst**.

---

# 💼 Resume Project Description

Built an **interactive Power BI dashboard analyzing 100K+ global data job postings** to identify salary trends, remote work availability, and job demand by location and platform.  
Used **Power Query, data modeling, and DAX** to transform raw job data into actionable insights including salary benchmarks and hiring trends.

---

# 📊 Business Questions Answered

This dashboard helps answer the following questions:

1. What is the **average salary for Data Analysts?**  
2. What **platforms post the most data jobs?**  
3. What **percentage of jobs allow remote work?**  
4. Which **job types dominate the market?**  
5. Where are **data jobs concentrated globally?**  
6. What is the **monthly trend of job postings?**

---

# 📈 Key Insights

- 💰 **Median Yearly Salary:** ~$90,000  
- ⏱ **Median Hourly Salary:** ~$32.50  
- 🏢 **LinkedIn** is the largest job posting platform  
- 🏠 ~**39% of jobs offer remote work**  
- 📄 **Full-time roles dominate (~88%)**  
- 🌍 Most data jobs are concentrated in **North America and Europe**

# 📁 Dataset Overview

The dataset contains **job posting information for data-related roles**, including salary details, job location, and platform sources.

### Columns Used

| Column | Description |
|------|-------------|
| company_name | Company offering the job |
| job_country | Country of job posting |
| job_health_insurance | Health insurance availability |
| job_location | Job location |
| job_no_degree_mention | Whether degree requirement is mentioned |
| job_posted_date | Date the job was posted |
| job_schedule_type | Job type (Full-time, Part-time) |
| job_skills | Skills mentioned in job |
| job_title | Job title |
| job_title_short | Simplified job title |
| job_type_skills | Skill category |
| job_via | Platform used for posting |
| job_work_from_home | Remote job indicator |
| salary_hour_avg | Average hourly salary |
| salary_rate | Salary type |
| salary_year_avg | Average yearly salary |
| search_location | Job search location |
